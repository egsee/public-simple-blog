    Go edit replace

    如果一个go模块尚未发布，我们无法直接使用模块路径进行导入。这时候可以使用go mod edit

    假设exmaple.com/hello依赖于example.com/greeting

    // 其中 ../greetings 为文件夹路径，可以与模块名不同
$ go mod edit -repalce example.com/greeting=../greetings
// 然后运行
$ go mod tidy
